This demo will showcase functionality that will allow the creation of metadata for use by the semantic video tool Popcorn.js. Popcorn allows for information to be pulled from various web sources and placed into a timeline to run alongside a playing HTML5 video. There are various types of metadata, each with their own fields. Click "Set Start" and "Set End" to set the start and end times in the timeline, and click "ADD" to add the data to the timeline

Existing chapter playing functionality remains in tact, but chapter creation has been taken out of this demo.
